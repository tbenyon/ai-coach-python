# Phase 4: Extension Tasks

Well done on completing the main phases! You now have a working fitness tracking app with AI-powered feedback.

Ready for more? The `extensions` folder contains additional challenges at different difficulty levels.

---

## Difficulty Levels

### Green (Easy)
Guided walkthroughs with full code examples. Follow the steps and you'll get it working.

### Amber (Medium)
Steps are provided but no code examples. You'll need to figure out the implementation yourself based on what you've learned.

### Red (Hard)
Minimal guidance - just the requirements and some hints. These are proper challenges that will test your understanding.

---

## Getting Started

1. Browse the `extensions` folder and pick a task that interests you
2. Start with green if you want more practice, or jump to amber/red if you're feeling confident
3. There's no set order - choose what sounds fun!

---

## Tips

- Look at existing code for patterns to follow
- Test frequently as you make changes
- If you get stuck, compare your code to similar working code in the project
- Ask for help if you need it!
